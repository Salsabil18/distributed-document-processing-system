# Distributed Intelligent Document Processing System using OCR

##  Overview

This project aims to develop a distributed platform for intelligent document processing using Optical Character Recognition (OCR).

The platform will allow users to upload scanned documents or images, extract text automatically, process multiple documents in parallel using distributed workers, and store the extracted information for search and management.

---

##  Objectives

- Extract text from images and PDF documents.
- Improve document quality before OCR.
- Process documents using distributed workers.
- Store extracted information in a database.
- Evaluate the performance of distributed processing.

---
